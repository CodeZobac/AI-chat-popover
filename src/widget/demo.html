Embeddable Chat Widget

This page demonstrates the standalone ETIC AI chatbot widget that can be embedded on any website. The widget is completely self-contained and uses Shadow DOM for style isolation.

Integration Instructions

To embed this widget on your website, add the following code before the closing </body> tag:

<script> window.EticAI = { config: { apiKey: 'your-api-key', // Optional theme: 'light', // 'light' or 'dark' position: 'bottom-right', // 'bottom-right' or 'bottom-left' primaryColor: '#2563eb', // Custom brand color allowedOrigins: ['https://yourdomain.com'], // Security autoOpen: false, // Auto-open on load showWelcomeMessage: true // Show welcome message } }; </script> <script src="https://etic-ai.vercel.app/widget/loader.js" async></script>

Features

  • Shadow DOM Isolation: Styles won't conflict with your website
  • Cross-Origin Support: Works on any domain with proper configuration
  • Responsive Design: Adapts to mobile and desktop screens
  • PostMessage API: Secure communication between domains
  • Customizable: Theme, colors, and positioning options
  • Lightweight: Minimal impact on page load performance
  • Accessibility: Full keyboard navigation and screen reader support

API Methods

Once loaded, you can control the widget programmatically:

// Initialize the widget EticAI.init({ theme: 'dark', position: 'bottom-left' }); // Update configuration EticAI.updateConfig({ primaryColor: '#10b981' }); // Destroy the widget EticAI.destroy();
